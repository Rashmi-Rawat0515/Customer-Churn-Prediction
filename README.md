# 📉 Customer Churn Predictor

A Machine Learning web app that predicts whether a customer will churn using Logistic Regression.

## What it does
- Trained on Telco customer data with SMOTE balancing
- Achieves ROC-AUC score of 0.84+
- Predicts churn probability with a visual risk bar
- Shows confusion matrix for model performance

## Tools used
Python, Scikit-learn, Logistic Regression, SMOTE, Pandas, Seaborn, Streamlit

## How to run
pip install -r requirements.txt
streamlit run app.py
